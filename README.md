<h2 align="center">Brain Tumor Detection using CNN</h2>

<p align="center">
  A deep learning approach for classifying brain MRI images as <b>tumor</b> or <b>no tumor</b>
  using a Convolutional Neural Network (CNN).
</p>

<p align="center">
  <b>Dataset:</b><br>
  <a href="https://www.kaggle.com/datasets/navoneel/brain-mri-images-for-brain-tumor-detection">
    Brain MRI Images for Brain Tumor Detection (Kaggle)
  </a>
</p>

<hr>

<h3>üß† Project Overview</h3>

<p>
This project implements a <b>Convolutional Neural Network (CNN)</b> using
<b>TensorFlow/Keras</b> to perform binary classification on brain MRI images.
The model learns discriminative features directly from image data and predicts
the presence or absence of a brain tumor.
</p>
